- Spark Streaming 기반 실시간 ETL
- 실시간 데이터 흐름 구조 학습
- Dashboard 연동 경험

# Pipeline Architecture

1. Producer가 이벤트 생성
2. Kafka Topic 적재
3. Spark Streaming 처리
4. 실시간 데이터 저장
5. Dashboard 시각화
