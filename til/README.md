# Today I Learned (TIL)

데이터 엔지니어 학습 내용을 날짜별로 기록하는 공간입니다.

---

# 목적

- 매일 학습한 내용 기록
- 문제 해결 과정 정리
- 트러블슈팅 문서화
- 꾸준한 학습 흐름 관리

```md
...

...
```
